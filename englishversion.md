# 🎧 JavaScript Practice Exercise | AudioBook  

This repository contains a practical JavaScript exercise developed during the first class of the JS intensive course by **Hashtag Treinamentos**. The goal of this exercise is to strengthen JavaScript skills through practical and interactive examples.  

---

## 📌 Overview  
In this project, you will find JavaScript code examples that cover fundamental concepts such as:  

- **DOM Manipulation**  
- **Event Handling**  
- **Control Structures**  
- **Functions**  

---

## 🌐 How to Access  
You can access the hosted version of the project **[by clicking here]([https://analurod.github.io/javascript-audiobook-challenge/](https://analurod.github.io/javascript-audiobook-challenge/))**.

---

## 📂 Project Structure  
The project is composed of the following files and directories:  

- 🗂 **`index.html`** → Main project page  
- 🎨 **`styles.css`** → CSS styles file  
- ⚡ **`script.js`** → JavaScript code with the exercises  
- 📄 **`README.md`** → Project documentation  
- 🖼 **`/images`** → Folder containing the images used in the project  
- 📚 **`/books`** → Audiobook files  

---

## 🚀 How to Run Locally  
To run this project on your local environment, follow the steps below:  

1. Clone this repository to your machine:  

   ```sh
   git clone https://github.com/analurod/Dom-Casmurro-Audiobook.git
   ```

2. Navigate to the project directory:
```
cd Dom-Casmurro-Audiobook
```

3. Open the ```index.html``` file in your preferred browser.
